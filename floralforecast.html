Floral Forecast

Summary

Garden/Farmer website for users to build their own garden and include information needed for the seed starting and growing process

Objectives

  • CSS
  • Layout using Bootstrap Grid
  • Using images for creative purposes
  • Created navbar using Bootstrap
  • Using CSS selectors
  • Include API's to pull in data for the user
  • Beekeper as a database to store the user garden information from the API
  • Fetched the data from database to website so the user can see their garden by using useState and useEffect
  • Fetched data from the Trefle botany API to bring in data for the user to learn more about gardening using useState and useEffect.
  • Fetched data from the NewsData.io API to bring in 10 gardening and flowers articles from around the world using useState and useEffect.
  • Fetched data from the USDA Farmers Market API so that the user can sell/find their product within 100 miles of their zipcode if needed using useState and useEffect.

Future Improvements

  • Add more growing information that the user can add to their gardens
  • Add a login feature
